Independence Jets
Simple animated jets that go around the screen showing Indian flag as smoke trails behind the jets. You can see them working here on this page or just try them on any website you like.


These jets can be implemented on any website, just include the jets.js file and initialize at the end of the document.

Include jets.js in <head>:

<script src="http://akshatmittal.github.io/independencejets/jets.js" />

And initialize just before (closing) </body>

<script>Independence.start("http://i.imgur.com/eoQQJet.png");</script>

Note that the image used for jets is specified when initializing, thus you can use whatever image you want. I found an image on the web and uploaded it to imgur, looks like this:



You can also stop them using JS by 'destroying' them, like this:

Independence.destroy();

You can create as many such instances as you want on a single page, though I don't recommend creating more than 2 on a page. This works on mobile, unlike this website.

You can use it freely on any website you like. I'll appreciate if you credit back with a link. Thanks.
